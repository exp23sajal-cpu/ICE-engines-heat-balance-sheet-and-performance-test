# IC Engine Performance & Heat Balance Analyzer

A Python-based computational tool for analyzing performance and heat balance 
of Single-Cylinder Petrol (SI) and Diesel (CI) engines using experimental observation data.

## Features
- Supports Petrol and Diesel engines
- Computes Brake Power, Indicated Power, BMEP
- Calculates Mechanical & Thermal Efficiency
- Generates Heat Balance Sheet
- Plots Engine Characteristic Curves

## Tech Stack
- Python
- Matplotlib
- Thermodynamics Modeling

## How to Run
```bash
pip install -r requirements.txt
python main.py
